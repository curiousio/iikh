# Interactive Intelligent Kitchen Helper

## Introduction

Interactive Intelligent Kitchen Helper (IIKH) is a simple CLI based meal planner written in C++ utilizing SQLite.

## Features

* Recipe management
* Plan management
* Meal management

## Requirements

* Microsoft Windows 11
* C++14
* SQLite3

## Installation

Clone this repository.

```shell
git clone https://github.com/curiousio/iikh.git
```

Checkout to the desired version.

```shell
git checkout v1.0.0
```

## Execution

Build from the source and run the application.

```shell
g++ main.cpp -l sqlite3
./a.out
```

Copy the `sqlite` directory into `Header Files` and the source code into `Source Files` if you are using Microsoft Visual Studio.
